#### Beweis i)
Sei es gibt keine Teilfolge, die gegen den limsup konvergeirt.\
Sei limsup = x\
Sei $b_n$ eine Teilfolge von $a_n$.\
Es gilt:\
$\forall \varepsilon >0: \forall  N\in\mathbb{N}: \exists n_0>N: |x-b_{n_0}|>x$
\
Dementsprechend müssen alle Elemente von a_n ab einem Bestimmten punkt diesen Abstand von x haben.\
Behauptun:\
Es muss einen kleinsten Abstand $\theta$ von x geben, sodass alle folgenden Elemente einen größeren Abstand von x haben.\
Beweis: \
Sei es gibt keinen kleinsten Abstand. Es kann dann eine Teilfolge gewählt werden, die gegen x konvergiert.
\
\
x ist dann nicht das lim sup von a, da auch $x-\theta$ größer ist als alle Elemente von a ab diesem PUnkt.\
Dies ist ein Widerspurch. \
somit gilt: \
Es muss eine Teilfolge geben, die gegen den lim sup konvergiert.
