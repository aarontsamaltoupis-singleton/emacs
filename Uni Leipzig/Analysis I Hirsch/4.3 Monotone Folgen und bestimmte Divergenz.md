# Definition 4.15: lim sup, inf sup
Sei $(a_{n})_{n}$ eine Folge reeller Zahlen. So definieren wir: 

Sei $b_{n}=\sup(\{ a_{m}: m\geq n \})$
Sei $c_{n}=\inf(\{ a_{m}: m\geq n \})$

Es gilt: 
$b_{n+1}\leq b_{n}$
$c_{n+1}\leq c_{n}$

$\lim\limits_{ n \to \infty }\sup a_{n}=\lim\limits_{ n \to \infty }\sup \{ a_{m}:m\geq n \}=\underset{n}{\inf}\underset{m\geq n}{\sup}a_{m}$
$\lim\limits_{ n \to \infty }\inf a_{n}=\lim\limits_{ n \to \infty }\inf \{ a_{m}:m\geq n \}=\underset{n}{\sup}\underset{m\geq n}{\inf}a_{m}$

Eine Folge $(a)_{n}$ konvergiert genau dann, wenn $\lim\limits_{ n }\inf a_{n} = \lim\limits_{ n } \sup a_{n}$

# Cauchy-Folgen/Cauchy-Kriterium
# Definition 4.16 (Cauchy-Folge) 
Eine Folge heißt Cauchyfolge, wenn sie das Cauchy-Kriterium erfüllt: 
$\forall\varepsilon>0:\exists N\in \mathbb{N}:|a_{n}-a_{m}|<\varepsilon, \forall n,m>N$

# Satz 4.17
Eine Folge $(a_{n})_{n}$ kovnergiert genau dann, wenn sie das Cauchykriterium erfüllt. 

Das Cauchykriterium ist also eine Möglichkeit zu überprüfen, ob eine Folge konvergiert, ohne den Grenzwert zu kennen.

## Bemerkung:
Ein [[metrische Raeume|metrischer Raum]] heiß$t$ vollständig, wenn jede Cauchyfolge konvergiert.
Das Cauchykriterium ist außerdem eine äquivalente Umformulierung des [[I Die reellen Zahlen#III Das Vollständigkeitsaxiom|Vollständigkeitsaxioms]].
