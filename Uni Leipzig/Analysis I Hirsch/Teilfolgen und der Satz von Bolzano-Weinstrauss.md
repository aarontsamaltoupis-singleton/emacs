# Definition 4.18 (Teilfolgen)
Sei $(a_{n})_{n}$ eine Folge und $(n_{k})_{k}$ eine aufsteigende ([[4.3 Monotone Folgen und bestimmte Divergenz#Definition 4.13 monoton fallend/wachsend|monotone]]) Folge in $\mathbb{N}$.
d.h. $n_{k}<n_{k+1}, \forall k$, dann heißt $(a_{n_{k}})_{k}$ **Teilfolge** von $(a_{n})_{n}$.

# Bemerkung: 
Wenn $a_{n}$ konvergent, dann ist auch jede Teilfolge konvergent
# Definition 4.19 (Häufungspunkt)
Sei $(a_{n})_{n}$ eine Folge, so heißt $a\in \mathbb{R}$ Häufungspunkt, falls es eine Teilfolge $(a_{n_{k}})_{k}$ gibt, mit $\lim\limits_{ n \to \infty }a_{n_{k}}=a$.

# Lemma 4.20
Sei $(a_{n})_{n}$ eine Folge , dann gilt: 
$a$ ist ein Häufungspunkt $\iff$ jeder [[4 Folgen, Reihen und Potenzwerte#Bemerkung 4.3 epsilon-Ball|epsilon-Ball]] um $a$ enthält unendlich viele Elemente.

# Satz 4.21 (Satz von Bolzano-Weierstrauß)
Sei $(a_{n})_{n}$ eine beschränkte Folge, dann existiert eine konvergente Teilfolge.
