# Definition 3.4.1 (Images of sets)
If $f:X\to Y$ is a [[Funktionen#Tao, Analysis I|function]] from $X$ to $Y$, and $S$ is a set in $X$, we define $f(S)$ to be the set 
$f(S):=\{ f(x):x\in S \}\subseteq Y$

This subset of Y is called the image of S under f.
f(S) is well defined  thanks to the [[Mengen#Tao#Axiom 3.6 (Replacement)|axiom of replacement]].

# Definition 3.4.4 (Inverse Image)
If $U\subseteq Y$ we deinfe the set $f^{-1}(U)$ to be the set
$f^{-1}(U):=\{x\in X:f(x)\in U  \}$

$f^{-1}(U)$ consists of all elements of x which map into U
$f(x)\in U\iff x\in f^{-1}$
