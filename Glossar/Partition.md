Die Partition einer [[Glossar/Mengen|Mengen]] $M$, ist eine Menge $P$, deren Elemente nichtleere Teilmengen von $M$ sind, sodass jedes Element von $M$ in genau einem Element von $P$ enthalten ist. 

Eine Partition $P$ einer Menge $M$ ist die Zerlegung dieser Menge in paarweise [[disjunkt|disjunkte]] Teilmengen.
