# Def 4.3.2 (Potenzreihen)
Eine Reihe der form
$P(x)=\sum\limits^{ \infty}_{n=0}a_{n}x^{n}=a_{0}+a_{1}x+a_{2}x_{2}+\dots$

heißt Potenzreihe, wobei $x\in \mathbb{R}$ (bzw. $x\in \mathbb{R}-\text{modul}$) und $(a_{n})_{n}$ ist eine reelle Folge.

# Lemma 4.33
Sei $P(x)$ konvergent in einem PUnkt $x_{0}$, dann ist $P(x)$ [[Konvergenzkriterien fuer Reihen|absolut konvergent ]] , für alle x mit $|x|<|x_{0}|$


# Def 4.34 (Konvergenzbereich und Konvergenzradius)
Sei $P(x)$ eine Potenzreie
So heißt die Menge $K=\{ x\in \mathbb{R}:P(x) \text{ konvergent} \}$ der **Konvergenzbereich** 
und $R = \sup\{ |x|: x\in K \}$ heißt **Konvergenzradius**

# Satz 4.35
Seien $P(x)=\sum\limits^{ \infty}_{n=0}a_{n}x^{n}$
 $Q(x)=\sum\limits^{ \infty}_{n=0}b_{n}x^{n}$
 zwei Potenzreihen, so gilt: 
 1. falss $P(x),Q(x_{})$ konvergent: $P(x)+Q(x_{})=\sum\limits^{ \infty}_{n=0}(a_{n}+b_{n})x^{n}$
 2. falls $P(x),Q(x)$ absolut konvergent: $P(x)Q(x)=\sum\limits^{ \infty}_{n=0}c_{n}x^{n}$ 
	 1. mit $c_{n}=\sum\limits^{ n}_{m=0}a_{n-m}b_{n}$




# Def 4.36 (Cauchy-Hadamard)
Sei $P(x) =\sum\limits^{}_{n}a_{n}x^{n}$ eine Potenzreihe, so gilt: 
$\frac{1}{R}=\limsup\sqrt[n]{ |a_{n}| }$
, wobei $R$ der Konvergenzradius von $P(x)$
