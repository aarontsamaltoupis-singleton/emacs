#Rudin

Eine komplexe Zahl ist ein [[geordnete Paare_ordered_pair|geordnetes Paar]] [[Der Körper der reellen Zahlen|reeller Zahlen]].

Seien $(a,b), (c,d)$ komplexe Zahlen

Wir definieren: 
$x+y= (a+c,b+d)$
$xy=(ac-bd, ad+bc)$

# Satz 1.25
Durch diese Definition der Addition und Multiplikation wird die Menge aller komplexen Zahlen zu einem [[Körper]] mit $(0,0)$ und $(1,0)$ in der Rolle von $0$ und $1$.

# Satz 1.26
Für alle reelle Zahlen $a,b$ gelten die Gleichheiten: 
$(a,0)+(b,0) = (a+b,0)$
$(a,0)(b,0)= (ab,0)$

# Definition 1.27: Definition von i
$i = (0,1)$

# Satz 1.29
$(a,b) = a+bi$

# Definition 1.30: Konjugierte
Sei $a,b\in \mathbb{R}$, Sei $z=a+bi\in \mathbb{C}$.
Die komplexe Zahl $\overline{z} =a-bi$ heißt die **Konjugierte** von z



# Satz 1.31
Seien $z$ und $w$ komplexe Zahlen. Dann gilt: 
- $|z|>0$,  außer für $z=0$
- $\overline{zw}=\overline{z}\cdot\overline{w}$
- $z+\overline{z} = 2\mathrm{Re}(z), z-\overline{z} = 2i\cdot\mathrm{Im}(z)$


# Definition 1.32
 Ist z eine komplexe Zahl, so ist ihr Betrag $|z|$ die nichtnegative Quadratwurzel von $z\overline{z}$, d.h. 
 $|z| = (z\overline{z})^{ \frac{1}{2{}}}$
